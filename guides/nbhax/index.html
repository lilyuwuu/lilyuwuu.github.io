New3DS new-browserhax Guide

This page will install Homebrew Launcher access without the use of movable.sed using the New 3DS / New 2DS Internet Browser. It can be used in conjunction with slotTool to install custom firmware. This guide will not work on an Old 3DS / Old 2DS.

Old 3DS users should be following 3ds.hacks.guide.

Before you start

This guide only works on New 3DS / New 2DS USA/EUR/JPN!! A New 3DS / New 2DS XL has a second circle pad (C-Stick) on the right-hand side, as well as ZL/ZR buttons on the back.

If you're following this guide to get CFW, please note that your left and right shoulder buttons must work in order to use slotTool. If either of those buttons are broken, you will still need to get movable.sed in order to follow alternate methods.

What you need

  • Your New Nintendo 3DS / New Nintendo 2DS XL console updated to the latest firmware (11.13.0-45 U/E/J)
    • This guide hypothetically works from 11.9/11.10 to 11.13, but there's no reason not to update
  • The latest release of new-browserhax
  • The latest release of Homebrew Launcher

Section I - Prep Work

  1. Copy boot.3dsx to the root of your SD card
    • The root of the SD card refers to the initial directory on your SD card where you can see the Nintendo 3DS folder, but are not inside of it
  2. Copy the contents of the folder corresponding to your console's region inside of the new-browserhax .zip (release_v1.0.zip) to the root of your SD card
    • arm11code.bin and browserhax_hblauncher_ropbin_payload.bin
    • If you are unsure of your console's region, you can find it in the version string in System Settings: U is USA, E is EUROPE, and J is JAPAN

Section II - new-browserhax

  1. Power on your device
  2. Open the Internet Browser (visual aid)
  3. Type https://zoogie.github.io/web/nbhax into the address bar
  4. Your device should eventually load into the Homebrew Launcher
    • If your device freezes on a yellow screen, try again
    • If your device freezes on a different coloured screen, or you see "An error has occurred", ensure that your file placement is correct and that you have selected the files corresponding to your console's region
