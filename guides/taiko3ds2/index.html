Taiko 3DS2 English Patch Guide

This page provides instructions to apply the English patch of Taiko no Tatsujin: Don to Katsu no Jikuu Daibouken ("Taiko no Tatsujin: Don and Katsu's Spacetime Adventure").
The guide that currently exists for this game was written in 2016, so the patching methods that it gives are no longer recommended. For example:

  • HANS is no longer used at all.
  • HackingToolkit9DS is no longer necessary.
  • LayeredFS and locale does not need to be set up through NTR.
  • Pre-patching CIAs is no longer preferable.

What this guide will do

This guide will:

  • Install the English patch of this game in a way where it can be easily disabled at will
  • Fix DLC if it is installed, hopefully

What you need

  • A clean copy of Taiko no Tatsujin: Don to Katsu no Jikuu Daibouken (cartridge or digitally installed)
  • Update 1.1 of the game already installed
  • The English patch (ROMFS English Menu Patch Version 1.1)
  • A modded Nintendo 3DS running modern CFW and tools (At the time of writing, the newest versions are Luma 10.2 and GodMode9 1.9.1)
    • This guide should technically work with Luma versions as old as 7.0 and GodMode9 versions as old as 1.2.7, but if you have an ancient setup, you should take some time to update it
  • The 0.04 nightly release of Luma Locale Switcher (LumaLocaleSwitcher-0.04-NIGHTLY.cia)

Section I - Prep Work

  1. Place LumaLocaleSwitcher-0.04-NIGHTLY.cia in the /cias/ folder
    • Create the folder if it does not already exist
  2. Extract ROMFS Version 1.1.zip to your working directory
    • The result should be several XDelta and patching files in a folder named ROMFS Version 1.1
  3. Put the SD card into your 3DS

Section IIa - Dumping RomFS (digital)

The instructions detailed here apply to digital copies of the game. If you have a cartridge, please follow Section IIb instead.

  1. Turn off the console
  2. Hold the START button while booting to enter GodMode9
  3. Press (A) to select the [A:] SYSNAND SD drive
  4. Navigate to title
  5. While 00040000 is highlighted, press (Right Shoulder) + (A)
  6. Select "Search for titles"
    • This may take a while if you have many titles
  7. Press (A) on 0004000000102F00 ... .tmd to select it
  8. Select "Open containing folder"
  9. Press (A) on 00000000.app to select it
  10. Select "NCCH image options..."
  11. Select "Mount image to drive"
  12. Press (A) to enter the path
  13. Press (A) on romfs.bin to select it
  14. Select "Copy to 0:/gm9/out"
    • This will take a while
  15. Transfer the romfs.bin file to the ROMFS Version 1.1 folder on your computer
    • Do not eject the SD card without first unmounting it

Section IIb - Dumping RomFS (cartridge)

The instructions detailed here apply to cartridge copies of the game. If you have already dumped the RomFS from a digital copy, proceed to Section III.

  1. Turn off the console
  2. Hold the START button while booting to enter GodMode9
  3. Insert the game cartridge
  4. Press (A) to select the [C:] GAMECART drive
  5. Press (A) to select 0004000000102F00_v0?.3ds
    • Trim/no trim does not matter
  6. Select "NCSD image options..."
  7. Select "Mount image to drive"
  8. Press (A) to enter the path
  9. Press (A) on romfs.bin to select it
  10. Select "Copy to 0:/gm9/out"
    • This will take a while
  11. Transfer the romfs.bin file to the ROMFS Version 1.1 folder on your computer
    • Do not eject the SD card without first unmounting it

Section III - Patching the game

  1. Rename romfs.bin to DecryptedRomFS.bin
    • Do not add the .bin extension if it does not appear
  2. Run the relevant xdelta script for your operating system
  3. You should now have a fullpatch v1.1_romfs.bin file

Section IV - Installing the patch

  1. Copy fullpatch v1.1_romfs.bin to any directory on your 3DS SD card
  2. Navigate to the file in GodMode9, and press (A) to select it
  3. Select "Mount as ROMFS image"
  4. Press (A) to enter the path
  5. Press (Left Shoulder) + (D-Pad Right) to mark all files
  6. Press (Y) to place all of these files in your clipboard
  7. Press (B) to enter the main drives menu
  8. Navigate to [0: SDCARD] -> luma -> titles -> 0004000000102F00 -> romfs
    • Create any folders that do not exist; there are sixteen characters in the Title ID
  9. Press (Y) to copy paths
    • This will take a while
  10. Exit GodMode9 by pressing the home button and selecting "Poweroff system"

Section V - Finalizing setup

  1. Turn on the console while holding SELECT to show the Luma Configuration menu
  2. Select the option Enable game patching
    • An (x) indicates that the option is enabled
  3. Press the START button to save the configuration
  4. Open the FBI application on your 3DS
  5. Navigate to the cias folder
  6. Press (A) to select LumaLocaleSwitcher-0.04-NIGHTLY.cia
  7. Select "Install and delete CIA"
  8. Exit the FBI application
  9. Open the Luma Locale Switcher application that is now on your home menu
  10. Press (A) To select "Titles"
  11. Select 太鼓の達人　どんとかつの時空大冒険
  12. Select "Change Region", and set the region to JPN
  13. Select "Change Language", and set the language to JP

The patch should now be enabled, and DLC should now work if installed.
